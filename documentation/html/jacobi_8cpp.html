Descripción detallada

Código en C++ que resuelve el algoritmo de Jacobi para sistemas lineales.

Omitiendo la lectura y escritura de datos, el algoritmo en pocas palabras sería tal que así:
DO
{
x1= x0 + noD*(b - A*x0);
x0= x1;
} WHILE(MAXIMO_ABSOLUTO(b - A*x0)>(umbralError*MAXIMO_ABSOLUTO(b)));

Siendo los datos de entrada:
MatrizCSR A:= Matriz de coeficientes
Array b:= vector del lado derecho
Array x0:= Aproximación inicial
long double umbralError:= Precisión de la solución que se quiere obtener (del roden de 1e-20)

Los datos que se calculan:
MatrizCSR noD:= Matriz con la diagonal de A negada
Array x1:= Solución del sistema

Funciones:
long double MAXIMO_ABSOLUTO(Array):= Devuelve el máximo absoluto de un vector.

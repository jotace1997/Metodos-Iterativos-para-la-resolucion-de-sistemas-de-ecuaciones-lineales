\doxysubsection{Descripción detallada}
Código en C++ que resuelve el algoritmo de Jacobi para sistemas lineales. 

Omitiendo la lectura y escritura de datos, el algoritmo en pocas palabras sería tal que así\+:~\newline
DO~\newline
 \{~\newline
 x1= x0 + no\+D$\ast$(b -\/ A$\ast$x0);~\newline
 x0= x1;~\newline
\} W\+H\+I\+LE(M\+A\+X\+I\+M\+O\+\_\+\+A\+B\+S\+O\+L\+U\+TO(b -\/ A$\ast$x0)$>$(umbral\+Error$\ast$\+M\+A\+X\+I\+M\+O\+\_\+\+A\+B\+S\+O\+L\+U\+TO(b)));~\newline
~\newline
 Siendo los datos de entrada\+:~\newline
\mbox{\hyperlink{classMatrizCSR}{Matriz\+C\+SR}} A\+:= Matriz de coeficientes~\newline
\mbox{\hyperlink{classArray}{Array}} b\+:= vector del lado derecho~\newline
\mbox{\hyperlink{classArray}{Array}} x0\+:= Aproximación inicial~\newline
long double umbral\+Error\+:= Precisión de la solución que se quiere obtener (del roden de 1e-\/20)~\newline
~\newline
 Los datos que se calculan\+:~\newline
\mbox{\hyperlink{classMatrizCSR}{Matriz\+C\+SR}} noD\+:= Matriz con la diagonal de A negada~\newline
\mbox{\hyperlink{classArray}{Array}} x1\+:= Solución del sistema~\newline
~\newline
 Funciones\+:~\newline
